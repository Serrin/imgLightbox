imgLightbox

Just another simple image lightbox without dependecies.

IE11, mobile browser and ES5 compatible.

API

Include

Insert the .js and .css file in the page header:

<link href="imglightbox.min.css" rel="stylesheet" type="text/css">
<script src="imglightbox.min.js"></script>

Insert this code in your html page:

<div id="imgLightbox" onclick="imgLightbox.close();">
  <div id="imgLightbox-button-area">
    <a title="open in new window and close" id="imgLightbox-button-newwindow">
      <span class="imgLightbox-button">&UpperRightArrow;</span>
    </a>
    <a title="download and close" id="imgLightbox-button-download">
      <span class="imgLightbox-button">&DownArrowBar;</span>
    </a>
    <span title="close" class="imgLightbox-button" onclick="imgLightbox.close();">&Cross;</span>
  </div>
</div>

Get the library version:

alert( imgLightbox.config.version );

Set config.download (default is false):

imgLightbox.config.download = true;
imgLightbox.config.download = false;

Set config.newwindow (default is false):

imgLightbox.config.newwindow = true;
imgLightbox.config.newwindow = false;

imgLightbox.open() method parameters:

imgLightbox.open(<filename: string>[,download: boolean[,newwindow: boolean]]);

Samples

Open with the download button:

imgLightbox.open('test1.jpg',true);

Small size image with the download button:

imgLightbox.open('test2.jpg',true);

Open without the download and with the newwindow buttons:

imgLightbox.open('test1.jpg',false,true);

Small size image without the download and with the newwindow buttons:

imgLightbox.open('test2.jpg',false,true);

Open with the download and newwindow buttons:

imgLightbox.open('test1.jpg',true,true);

Small size image with the download and newwindow buttons:

imgLightbox.open('test2.jpg',true,true);

Open with the config.download and config.newwindow settings:

imgLightbox.open('test1.jpg');

Small size image with the config.download and config.newwindow settings:

imgLightbox.open('test2.jpg');

Open without the download button:

imgLightbox.open('test1.jpg',false);

Small size image without the download button:

imgLightbox.open('test2.jpg',false);

Open without the download and newwindow buttons:

imgLightbox.open('test1.jpg',false,false);

Small size image without the download and newwindow buttons:

imgLightbox.open('test2.jpg',false,false);
